## Women Safety and Complaint Management System

# Overview
This project is a web-based Women Safety and Complaint Management System developed using Flask and Oracle Database. It provides a structured platform for users to register complaints, track their status, and submit feedback after resolution. The system includes separate modules for users, admin, and safety officers.

# Features

User:
- Register and login
- File complaints
- View complaint status (Pending, In Progress, Resolved)
- Submit feedback after resolution

Admin:
- View all users and officers
- Assign complaints to officers
- Monitor all complaints

Safety Officer:
- View assigned complaints
- Update complaint status

# Tech Stack
- Frontend: HTML, CSS
- Backend: Python (Flask)
- Database: Oracle Database

# Database
- Relational schema with normalization
- Primary key and foreign key constraints
- Ensures data consistency and integrity

# System Workflow
1. User registers and logs in
2. User files a complaint
3. Admin assigns complaint to an officer
4. Officer updates the complaint status
5. User tracks status and submits feedback
